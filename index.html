🔗 Setup JSONBin.io (sync tim)

1

Buka jsonbin.io → daftar gratis

2

Klik New Bin → paste {"tasks":[],"products":[]} → Create

3

Copy Bin ID dari URL (angka setelah /b/)

4

Klik API Keys → Create Access Key → centang Read + Update → copy key

5

Bagikan Bin ID & Key yang sama ke semua tim → input saat setup
